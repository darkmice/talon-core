# Talon SDK

SDK 源码已迁移至独立仓库，请前往：

**https://github.com/darkmice/talon-sdk**

## 支持语言

| 语言 | 目录 | 绑定方式 |
|------|------|---------|
| Go | `go/` | cgo (静态链接) |
| Python | `python/` | ctypes |
| Node.js | `nodejs/` | ffi-napi |
| Java | `java/` | JNA |
| .NET | `dotnet/` | P/Invoke |

## 支持平台

macOS (arm64/amd64) · Linux (amd64/arm64/loongarch64/riscv64) · Windows (amd64)

含信创平台：鲲鹏、飞腾、海光、兆芯、龙芯、RISC-V。
